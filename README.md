# Adondevamos.web v.Alpha

## Project Overview
Adondevamos.web is a front end project of the website Adondevamos.io made on REACT

## Goal
Create trips to share place recomendations and vote to help to decide ¿A donde vamos?

## Features
- Create a list of places you want to go at a trip
- Add members to colaborate with recomendations, vote and help you manage this trip
- Help others to think about countrys or places
- Ranking of most voted places, countries and trips

## Prerequisites
- Node.js (v14.x or higher recommended)
- npm (v6.x or higher) or yarn

## Installation
```bash
git clone https://github.com/your-username/your-repo-name.git
cd adondevamosweb
npm install
```
